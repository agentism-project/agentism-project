# Governance

This directory contains the governance framework for the Agentism project —
**Durable (Layer 2)** documents that define how decisions are made,
who holds what roles, and how the community is organized.

## Layer 2: Durable

Governance documents are **Layer 2 (Durable)**. Changes require:
- A formal RFC (open issue with full proposal)
- 30-day public discussion period
- >60% majority vote (quorum: 30% of active contributors)
- 14-day cooling period

See [decision-making.md](decision-making.md) for the full process.

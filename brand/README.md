# Agentism Brand Assets

This directory contains the official visual identity of the Agentism Project.

All assets are published under [CC BY-SA 4.0](../LICENSE-docs), consistent with the project's open licensing (CC-041). Trademark protections for the name "Agentism" and the symbol still apply — see [TRADEMARK.md](../TRADEMARK.md).

## Quick Reference

| Asset | Format | Use |
|-------|--------|-----|
| `agentism-symbol.svg` | SVG (vector) | Primary use — website, documents, print |
| `agentism-symbol-monochrome.svg` | SVG (vector) | Single-color printing, stamps, embossing |
| `agentism-symbol-inverted.svg` | SVG (vector) | Dark backgrounds |
| `agentism-favicon.svg` | SVG (vector) | Browser favicon, app icon |

## PNG Exports

PNG raster exports at standard sizes (16px, 32px, 64px, 128px, 256px, 512px, 1024px) are generated from the SVG source files. See [brand-usage-guide.md](brand-usage-guide.md) for export instructions.

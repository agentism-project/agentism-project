title: "Agentism"
description: "Productive agents for universal human well-being."
---

Agentism designs the transition to a society where autonomous agents generate broad prosperity — and that prosperity is shared equitably with all human beings.

This project is in **Phase 0: Foundation**. The core principles and governance framework are being established. We welcome input from all contributors.

[How to Participate](guides/how-to-participate/) — no technical knowledge required.

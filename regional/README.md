# Regional Adaptation Framework

This directory contains regional adaptations of the Agentism framework —
documents produced by regional communities translating universal principles
into locally viable proposals.

## How Regional Adaptation Works

The Agentism framework operates on three layers:

```
UNIVERSAL LAYER (Layer 1 — Immutable)
  Five core principles (CP-001 to CP-005)
  Apply everywhere. Cannot be overridden regionally.

ADAPTATION LAYER (Layer 2 — Durable)
  Regional working groups translate universal principles
  into locally viable policy proposals.

IMPLEMENTATION LAYER (Layer 3 — Adaptive)
  Actual policy advocacy, pilot projects, and community
  organizing, carried out by local actors.
```

Each regional adaptation must:
1. Reference which universal principle(s) it serves
2. Explain why the specific approach fits the region
3. Identify local stakeholders consulted
4. Include local success/failure metrics
5. Self-assess current Transition Level (0–3) with supporting evidence

## Regional Adaptation Template

A template for creating regional adaptations is in the `templates/`
subdirectory. See the full requirements in
[SRS Section 10](../SRS/10-regional-adaptation.md).

## Transition Levels

| Level | Name | Description |
|-------|------|-------------|
| 0 | Pre-Agentist | Automation without redistribution or ethical regulation |
| 1 | Local Initiatives | Minimal redistribution mechanisms, pilot programs |
| 2 | Institutional Frameworks | Ethical governance and auditing at sector level |
| 3 | Agentist Society | Structural redistribution, post-labor citizenship |

Regional adaptations propose realistic next-level targets — one level
up from current state, with specific milestones.

## Current Adaptations

*No regional adaptations have been published yet. Phase 0 focuses on
the global framework. Regional adaptations are a Phase 1 deliverable.*

To propose a regional adaptation, use the template in `templates/`
and open a GitHub Discussion in your regional working group category.
